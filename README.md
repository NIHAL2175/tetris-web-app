## 🧠 Project Overview

This project contains two versions of a browser-based Tetris game built using React. The application demonstrates modern frontend development concepts including component-based architecture, responsive design, state management, and cloud deployment. Players can enjoy the classic falling-block puzzle experience directly in their web browser.

## 🎮 Application — Tetris V1

### 📌 What is Tetris V1?

Tetris V1 is the initial implementation of the game. It focuses on delivering the classic Tetris experience with essential gameplay mechanics, responsive controls, and score tracking functionality.

### 🗂️ Directory Structure

```text
Tetris-V1/
 ├── public/
 ├── src/
 │   ├── components/
 │   ├── helpers/
 │   └── tetris/
 ├── Dockerfile
 ├── package.json
 └── package-lock.json
```

### ⚙️ How It Works

* Built using React.js
* Handles falling tetromino pieces
* Detects completed rows
* Updates score dynamically
* Provides responsive gameplay controls

### 🎯 Key Features of V1

* ✅ Classic Tetris gameplay
* ✅ Score tracking
* ✅ Responsive design
* ✅ Browser-based experience
* ✅ Lightweight implementation
